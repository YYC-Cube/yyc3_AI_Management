## 一、项目概述

### 1.1 项目定位

**YanYu Cloud³ (YYC³)** 是一个企业级 **AI驱动的智能业务管理平台**，采用前后端分离架构，集成工作流引擎、AI智能分析、实时通信等核心能力，为企业提供数字化转型的一站式解决方案。

### 1.2 核心价值主张

| 维度 | 价值主张 | 实现方式 |
|------|----------|----------|
| **智能化** | AI驱动决策 | OpenAI GPT-4o集成、智能分析引擎 |
| **流程化** | 工作流自动化 | 可视化流程设计器、多级审批 |
| **实时化** | 即时响应 | WebSocket双向通信、Redis缓存 |
| **安全化** | 企业级防护 | JWT认证、RBAC权限、数据加密 |
| **标准化** | 规范输出 | YYC³五高五标五化体系 |

### 1.3 目标用户

- **企业管理者**: 需要一体化业务管理平台的中大型企业
- **开发团队**: 需要快速搭建管理系统开发框架的技术团队
- **运维人员**: 需要完善监控和运维保障的IT团队
- **决策者**: 需要数据支撑和智能分析的管理层

## 六、快速开始

### 6.1 环境要求 (最低)

- ✅ Node.js >= 18.0.0 (LTS推荐)
- ✅ npm >= 9.0.0 或 pnpm >= 8.0.0
- ✅ PostgreSQL >= 15.0
- ✅ Redis >= 7.0
- ✅ Git (最新版本)

### 6.2 三步启动

```bash
# Step 1: 克隆仓库
git clone https://github.com/YYC-Cube/yyc3-AI-Management.git
cd yyc3-AI-Management

# Step 2: 安装依赖
npm install && cd backend && npm install && cd ..

# Step 3: 配置并启动
cp .env.example .env.local
# 编辑 .env.local 填入实际配置
npm run dev  # 启动前端 (端口3000)
cd backend && npm run dev  # 启动后端 (端口3001)
```

> 📖 详细环境配置请参阅: [003-API-Mana-项目总览索引-快速开始指南.md](./003-API-Mana-项目总览索引-快速开始指南.md)
